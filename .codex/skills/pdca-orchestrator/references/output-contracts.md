# Output Contracts

Keep phase outputs compact.

## Plan
- Objective
- Scope
- Constraints
- Assumptions
- Success Criteria
- Next Step

## Design
- Options Considered
- Chosen Approach
- Target Files
- Validation Plan
- Out of Scope

## Do
- Changes Made
- Validation Run
- Open Issues

## Analyze
- Outcome
- Verified
- Risks
- Next Action

Use prose unless the content is inherently list-shaped.
