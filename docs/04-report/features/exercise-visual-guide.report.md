# 기능 보고서: 운동 시각 자료 안내

**기능명:** `exercise-visual-guide`
**버전:** 1.0.0
**날짜:** 2026-04-01

## 1. 요약

'운동 시각 자료 안내' 기능이 성공적으로 구현되어 애플리케이션에 통합되었습니다. 이 기능은 사용자에게 운동 동작에 대한 시각 자료(GIF 또는 유튜브 썸네일)를 제공하여, 올바른 자세를 이해하고 안전하게 운동을 수행할 수 있도록 돕습니다.

## 2. 구현 세부 정보

-   **데이터베이스:** 데이터베이스 마이그레이션을 통해 `exercises` 테이블에 `visual_guide_url` (`TEXT` 타입) 열이 추가되었습니다.
-   **데이터 타입:** `Exercise` 및 `ProgramExerciseRow` 타입이 `visual_guide_url` 필드(선택 사항)를 포함하도록 업데이트되었습니다.
-   **상태 관리:** `workout-store`가 프로그램으로부터 운동 세션을 생성할 때 새로운 `visual_guide_url` 필드를 올바르게 처리하도록 업데이트되었습니다. 이제 검색 기능도 이 URL을 정상적으로 가져옵니다.
-   **프론트엔드 컴포넌트:** 재사용 가능한 새로운 컴포넌트 `ExerciseVisualGuide.tsx`가 생성되었습니다. 이 컴포넌트는 GIF를 직접 표시하거나 유튜브 영상의 클릭 가능한 썸네일을 보여줄 수 있습니다.
-   **UI 통합:** 새로운 컴포넌트는 다음 두 핵심 화면에 통합되었습니다:
    -   `workout-session-screen.tsx`: 진행 중인 운동 화면에서 운동 이름 아래에 시각 자료가 표시됩니다.
    -   `program-detail-screen.tsx`: 프로그램 상세 정보 화면의 일일 운동 목록에서 운동 이름 옆에 시각 자료가 표시됩니다.

## 3. 테스트 방법

1.  **데이터베이스 마이그레이션 적용:** 새로운 마이그레이션 파일 `supabase/migrations/20260401_add_visual_guide_to_exercises.sql`을 실행하여 데이터베이스 스키마를 업데이트하세요.
2.  **데이터 채우기:** `exercises` 테이블에 있는 하나 이상의 운동에 대해 `visual_guide_url` 열에 URL을 수동으로 추가하세요. GIF URL 또는 유튜브 영상 URL을 사용할 수 있습니다.
    -   예시 GIF URL: `https://i.imgur.com/some-gif.gif`
    -   예시 유튜브 URL: `https://www.youtube.com/watch?v=some-video-id`
3.  **프로그램 상세 화면에서 테스트:** 시각 자료 URL이 있는 운동을 포함하는 프로그램 상세 화면으로 이동하세요. 운동 이름 옆에 안내 자료(썸네일 또는 GIF)가 나타나야 합니다.
4.  **운동 세션에서 테스트:** 시각 자료 URL이 있는 운동을 포함하는 운동 세션을 시작하세요. 운동 카드에 안내 자료가 표시되어야 합니다. 유튜브 썸네일을 클릭하면 링크가 열려야 합니다.

## 4. 향후 개선 가능성

-   **관리자 인터페이스:** 데이터베이스에 직접 접근하지 않고도 시각 자료 URL을 쉽게 추가/업데이트할 수 있는 간단한 관리자 인터페이스를 만듭니다.
-   **비디오 플레이어:** 더 원활한 경험을 위해 유튜브 앱으로 연결하는 대신 인앱 유튜브 플레이어를 구현합니다.
-   **콘텐츠 수집:** 새로운 운동에 대한 시각 자료 수집을 자동화합니다.
-   **성능:** GIF의 경우, 설계에서 제안된 대로 `react-native-fast-image`를 사용하여 로딩 및 캐싱을 더욱 최적화합니다.
