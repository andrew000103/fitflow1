# FitFlow MVP IA

## Primary Bottom Tabs

- Community
- History
- Train
- Analytics
- Profile / Me

## Secondary Flow

- Nutrition
  - Initial MVP: accessed from `Analytics`, `Profile`, and dashboard-style widgets
  - Expansion phase: promoted to a primary bottom tab when nutrition DAU justifies it

## Screen Tree

### Community

- Feed home
- Shortform viewer
- Post detail
- Comments
- Goal-based recommendation feed

### History

- Daily history timeline
- Workout logs
- Nutrition logs
- Exercise-specific history

### Train

- Today routine
- Exercise picker
- Set logger
- Rest timer
- Exercise detail and tutorial link

### Analytics

- Weekly progress dashboard
- PR and volume trends
- Fatigue and recovery
- Calorie summary
- Nutrition hub entry

### Profile / Me

- Goal settings
- Body metrics
- Notification settings
- Recommendation settings
- Nutrition hub entry

### Nutrition Secondary Flow

- Quick meal logging
- Search and saved foods
- Daily calorie summary
- Intake vs burn vs net review
